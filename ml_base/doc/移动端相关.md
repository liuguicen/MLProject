# 量化
## 动态量化：
它目前只支持 nn.Linear 和 nn.LSTM，这意味着您必须使用静态量化或量化感知训练(稍后将讨论)来量化其他模块，如 nn.Conv2d.
静态 需要训练 样本层次和通道层次 通道层次精度高很多
感知 要训练，就是低精度训练，损失很小
